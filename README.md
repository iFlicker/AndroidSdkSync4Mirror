## AndroidSdkSync4Mirror

Android SDK Sync for Mirror.site  
一个为镜像站所用的Android SDK 更新脚本  
(话说境外也用不上这玩意儿,就没英文说明了)

* * *

## 功能

*   支持从国内其他镜像站更新'/Android/repository/'下的数据
*   支持定时检测上游源是否有更新
*   支持日志管理
*   支持配置文件
